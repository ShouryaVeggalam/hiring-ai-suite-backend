# File & Folder Guide

## Root (`backend/`)

| File | Purpose |
|------|---------|
| `package.json` | Dependencies and npm scripts |
| `tsconfig.json` | TypeScript compiler options |
| `jest.config.ts` | Jest test runner config |
| `.env.example` | Documented environment template |
| `Dockerfile` | Multi-stage production image |
| `docker-compose.yml` | API, worker, Postgres, Redis, MinIO |
| `README.md` | Setup and phase status |

## `src/config/`

| File | Purpose |
|------|---------|
| `env.ts` | Zod schema — app fails fast on invalid env |
| `index.ts` | Cached config accessor |
| `logger.ts` | Pino logger factory |
| `prisma.ts` | PrismaClient singleton |
| `redis.ts` | ioredis client |
| `queue.ts` | BullMQ connection + queue name constants |
| `container.ts` | tsyringe DI registrations |
| `swagger.ts` | OpenAPI spec builder |

## `src/database/prisma/`

| File | Purpose |
|------|---------|
| `schema.prisma` | All models, enums, relations, indexes |
| `seed.ts` | Demo org + admin user |
| `migrations/` | Generated by `prisma migrate` |

## `src/app.ts` / `src/server.ts`

| File | Purpose |
|------|---------|
| `app.ts` | Express app: middleware, routes, errors |
| `server.ts` | HTTP listen + graceful shutdown |

## `src/middleware/`

| File | Purpose |
|------|---------|
| `requestId.middleware.ts` | Correlation ID per request |
| `rateLimiter.middleware.ts` | Global + auth rate limits |
| `auth.middleware.ts` | JWT stub (Phase 2) |
| `role.middleware.ts` | RBAC guard |
| `tenant.middleware.ts` | Sets org context from user |
| `validate.middleware.ts` | Zod validation helper |
| `error.middleware.ts` | Centralized error JSON |
| `notFound.middleware.ts` | 404 handler |

## `src/controllers/` & `src/routes/`

HTTP adapters and route maps. Phase 1 includes only `health.*`.

## `src/services/` & `src/repositories/`

Business logic and Prisma access. Populated in Phases 2–6.

## `src/providers/ai/`

| File | Purpose |
|------|---------|
| `IScreeningProvider.ts` | Screening port |
| `IQuestionProvider.ts` | Questions port |
| `IComparisonProvider.ts` | Comparison port |
| `mock/mockScreening.provider.ts` | Deterministic mock AI |
| `provider.factory.ts` | Provider selection by env |

## `src/storage/`

S3 abstraction (`IStorage`, factory). Implemented in Phase 3.

## `src/queues/`

| File | Purpose |
|------|---------|
| `index.ts` | BullMQ queue factories |
| `workers.ts` | Worker process entry (Phase 3+) |

## `src/types/` & `src/utils/`

Shared TypeScript types, AI contracts, `ApiError`, `ApiResponse`, pagination.

## `tests/`

Jest unit, integration, and e2e tests (Phase 7+).

## `docs/`

Architecture, database ER, OpenAPI, and this structure guide.
